This project creates a 3D plane, where you can drag and place Lego-style blocks.
It's designed to work like a very-simplified version of Sketchup.
There are 3 axes. You can orbit, pan, and zoom.
And the program attempts to guess which axis you are moving along, like Sketchup.

Live demo: http://www.jonathankoomjian.com/projects/lego/

https://github.com/jkoomjian/Lego-Sketchup
